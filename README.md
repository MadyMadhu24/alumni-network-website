# Alumni Network Website

## Overview
This project is a frontend-based Alumni Network Website developed during my internship at SITER Academy (Norway).

The website provides an interactive platform where alumni can connect, view updates, participate in events, and explore networking opportunities.

## Features
- Responsive Design
- Alumni Dashboard
- Posts & Updates
- Events Section
- Calendar
- Clock Widget
- Suggested Connections
- Messaging Interface
- Mentorship Programs
- Like, Comment & Share UI
- User-Friendly Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Files
- `index.html` – Landing page
- `dashboard.html` – Alumni Dashboard
